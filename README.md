I've done my best here to create profiles designed for dungeons & casual play. 

* **No more blowing your 2-minute burst** on a boss with 1% HP left!
* **No more manually turning off DoTs** when there are 9 enemies around you.
* **No more worrying about mitigations** either never getting used or getting used improperly.

These reactions should work well in all daily roulettes, treasure maps, hunts/fates, overworld combat, and whatever else you're doing that isn't raiding. 

These profiles took many months of testing and iteration to develop. I share them freely because I enjoy contributing to the community. If you build upon this work, all I ask is that you please acknowledge where it came from.

## Core Toggle Features
There are at least 3 toggles available for each job:
* **Mitigation:** Uses every part of your job's mitigation kit automatically on incoming raidwide attacks.
* **Variable Holds:** Holds your cooldowns to ensure full 2-minute bursts are ready before a boss dies.
* **QOL Tweaks:** General quality-of-life adjustments (such as out-of-combat sprint).

## Job Features

### 🛡️ Tank
* Full dungeon auto-mitigation
* Variable holds
* Automated Buddy-Mitigation on the lowest HP ally
* Auto-stun targets in enemy mobs
* Out-of-combat sprint
* Auto-burn last boss for expert maps

### ⚔️ Melee DPS
* Auto-Feint
* Personal mitigation & party mitigation
* Variable holds
* DoT manager (where applicable)
* Out-of-combat sprint
* Auto-burn last boss for expert maps
* **Job Specifics:** Samurai *Hagakure* out-of-combat optimization

### 🏹 Physical Ranged DPS
* Automatic mitigations
* Variable holds
* Out-of-combat sprint
* Toggle resets automatically when exiting combat
* Auto-burn last boss for expert maps
* **Job Specifics:** 
  * Bard *Esuna* management
  * Bard automatic song management for dungeons
  * Bard DoT manager
  * Dancer *Curing Waltz* distance & HP threshold checks

### 🪄 Caster DPS
* Auto-Addle
* Auto personal & party mitigation
* Variable holds
* Out-of-combat sprint
* Auto-burn last boss for expert maps
* **Job Specifics:**
  * Black Mage DoT manager
  * Summoner *Rekindle* on the main tank

### 🟢 Healer
* Auto GCD & oGCD healing
* Auto-mitigations
* Variable holds
* DoT manager
* Out-of-combat sprint
